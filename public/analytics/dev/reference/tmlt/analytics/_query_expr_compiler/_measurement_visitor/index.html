visit_groupby_quantile(self, query)#

Create a measurement from a GroupByQuantile query expression.

This method also checks to see if the schema allows invalid values (nulls, NaNs, and infinite values) on the measure column; if so, the query has DropNullAndNan and/or DropInfinity queries inserted immediately before it is executed.
