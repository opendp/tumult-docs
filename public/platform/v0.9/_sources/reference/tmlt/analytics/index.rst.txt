.. py:module:: tmlt.analytics

.. autoapi-nested-parse::

   Tumult Analytics is a differentially private analytics library from `Tumult Labs`_.

   .. _Tumult Labs: https://tmlt.io

   The library is broken up into a number of modules that provide different
   functionality. These modules fall into a few broad categories, described below.

   **For specifying privacy guarantees**:

   * :mod:`~.session` defines the :class:`~.session.Session`, an interactive
     interface for making differentially private queries.

   * :mod:`~.privacy_budget` contains types for representing privacy budgets.

   * :mod:`~.protected_change` contains types for expressing what changes in input
     tables are protected by differential privacy.

   **For defining queries**:

   * :mod:`~.query_builder` provides an interface for constructing differentially
     private queries from basic query operations.

   * :mod:`~.keyset` defines :class:`~.keyset.KeySet`, a type for specifying the
     keys used in group-by operations.

   * :mod:`~.constraints` contains types for representing constraints used when
     evaluating queries on data with the :class:`~.protected_change.AddRowsWithID`
     protected change.

   * :mod:`~.truncation_strategy` and :mod:`~.binning_spec` provide types that are used
     to define certain types of queries.

   |PRO| **For evaluating error**:

   * :mod:`~.program` provides an interface for defining a particular differentially
     private computation as a reusable program object.

   * :mod:`~.tuner` provides an interface for evaluating and tuning the error of a program.

   * :mod:`~.metrics` defines a collection of metrics which can be used as part of
     the tuning process.


Modules
=======

.. list-table::

   * - :mod:`.binning_spec`
     - A BinningSpec defines a binning operation on a column.
   * - :mod:`.constraints`
     - Defines :class:`~tmlt.analytics.constraints.Constraint` types.
   * - :mod:`.keyset`
     - A KeySet specifies a list of values for one or more columns.
   * - :mod:`.metrics`
     - |PRO| Metrics to measure the quality of program outputs.
   * - :mod:`.no_privacy_session`
     - |PRO| Interactive query evaluation without any privacy guarantees.
   * - :mod:`.privacy_budget`
     - Classes for specifying privacy budgets.
   * - :mod:`.program`
     - |PRO| SessionProgram and SessionProgramBuilder interfaces.
   * - :mod:`.protected_change`
     - Types for programmatically specifying what changes in input tables are protected.
   * - :mod:`.query_builder`
     - An API for building differentially private queries from basic operations.
   * - :mod:`.query_expr`
     - Building blocks of the Tumult Analytics query language. Not for direct use.
   * - :mod:`.session`
     - Interactive query evaluation using a differential privacy framework.
   * - :mod:`.truncation_strategy`
     - Defines strategies for performing truncation in private joins.
   * - :mod:`.tuner`
     - |PRO| Interface for tuning SessionPrograms.
   * - :mod:`.utils`
     - Utility functions.
