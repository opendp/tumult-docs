Deployment & troubleshooting
============================

The Tumult Platform is a Python library. It can be installed either
:ref:`locally <installation>`, on a server (including in tightly controlled or
air-gapped environments), or on a Cloud platform.

The following guides provide more details on how to deploy and run the Tumult Platform
in various environments. Additional pages focus on
:ref:`Spark configuration and performance <spark>`, as well as :ref:`Troubleshooting`.
