The API of the Tumult Platform is split in three distinct components.

* :ref:`Tumult Analytics<analytics-api>` allows users to compute statistical
  queries on tabular data using differential privacy. The Tumult Analytics API also
  includes the fundamental abstractions used throughout the Tumult Platform to capture
  privacy
  guarantees.
* :ref:`Tumult Synthetics<synthetics-api>` allows users to generate differentially
  private synthetic data.
* :ref:`Tumult Tune<tune-api>` allows users to measure and optimize the accuracy
  of differentially private programs.

.. _analytics-api:

Tumult Analytics
----------------

The Tumult Analytics API reference is split in
three
major sections.

* :ref:`Specifying privacy guarantees<privacy-guarantees>` introduces the
  :class:`~tmlt.analytics.Session`, the fundamental abstraction used to enforce
  differential privacy guarantees on sensitive data. It also lists classes used to
  define privacy budgets and protected changes.
* :ref:`Building queries<queries>` introduces the classes and methods used to
  define statistical queries to evaluate with differential privacy.

* |PRO| :ref:`Defining and using programs<programs>` introduces the
  :class:`~tmlt.analytics.SessionProgram`, a way to define structured differential
  privacy programs that can be used with higher-level tools, such as Tumult Tune.

Two further pages list :ref:`configuration options<config>` and :ref:`utility
functions<utils>`.

.. _synthetics-api:

|PRO| Tumult Synthetics
-----------------------

The Tumult Synthetics API reference lists classes and methods to
:ref:`generate synthetic data<Generating synthetic data>`.

.. _tune-api:

|PRO| Tumult Tune
-----------------

The Tumult Tune API reference is split in two sections.

* :ref:`Tuning programs<tuners>` introduces the :class:`~tmlt.tune.SessionProgramTuner`,
  the main class used to generate error reports. It also lists the various ways that
  users can interact with error reports.
* :ref:`Defining error metrics<metrics>` lists the built-in error metrics provided by
  Tumult Tune, and the methods and classes used to define custom error metrics.
