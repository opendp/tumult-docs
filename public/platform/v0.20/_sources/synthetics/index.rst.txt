Tumult Synthetics documentation
===============================

.. note:: 

    |PRO_NOTE|

Tumult Synthetics is a Python library for generating synthetic tabular data with
differential privacy.

New users should start with the :ref:`installation instructions <installation>`, then
follow the :ref:`tutorial <synthetic-data-basics>`.
