*********************
tmlt.analytics
*********************

.. py:module:: tmlt.analytics

.. autoapi-nested-parse::

   `Tumult Labs' <https://tmlt.io>`_ differentially private analytics library.


Modules
=======

.. list-table::

   * - :mod:`.binning_spec`
     - A BinningSpec defines a binning operation on a column.
   * - :mod:`.constraints`
     - Defines :class:`~tmlt.analytics.constraints.Constraint` types.
   * - :mod:`.keyset`
     - A KeySet specifies a list of values for one or more columns.
   * - :mod:`.metrics`
     - |PRO| Metrics to measure the quality of program outputs.
   * - :mod:`.no_privacy_session`
     - |PRO| Interactive query evaluation without any privacy guarantees.
   * - :mod:`.privacy_budget`
     - Classes for specifying privacy budgets.
   * - :mod:`.program`
     - |PRO| SessionProgram and SessionProgramBuilder interfaces.
   * - :mod:`.protected_change`
     - Types for programmatically specifying what changes in input tables are protected.
   * - :mod:`.query_builder`
     - An API for building differentially private queries from basic operations.
   * - :mod:`.query_expr`
     - Building blocks of the Tumult Analytics query language. Not for direct use.
   * - :mod:`.session`
     - Interactive query evaluation using a differential privacy framework.
   * - :mod:`.truncation_strategy`
     - Defines strategies for performing truncation in private joins.
   * - :mod:`.tuner`
     - |PRO| Interface for tuning SessionPrograms.
   * - :mod:`.utils`
     - Utility functions.
