.. py:class:: Catalog

   Specifies schemas and constraints on public and private tables.






   .. automethod:: __init__
   .. py:method:: add_private_table(source_id, col_types, grouping_column = None, id_column = None, id_space = None)

      Adds a private table to catalog. There may only be a single private table.

      :param source_id: The source id, or unique identifier, for the private table.
      :param col_types: Mapping from column names to types for private table.
      :param grouping_column: Name of the column (if any) that must be grouped by in any
                              groupby aggregations that use this table.
      :param id_column: Name of the ID column for this table (if any).
      :param id_space: Name of the identifier space for this table (if any).

      :raises ValueError: If there is already a private table.


   .. py:method:: add_public_table(source_id, col_types)

      Adds public table to catalog.

      :param source_id: The source id, or unique identifier, for the public table.
      :param col_types: Mapping from column names to types for the public table.

      :raises ValueError: If there is already a private table.


   .. py:method:: tables()
      :property:

      Returns the catalog as a dictionary of tables.
