.. py:class:: QuantileAbsoluteError(output, measure_column, quantile, join_columns, *, name = None, description = None, baselines = None)

   Bases: :class:`tmlt.analytics.metrics._base.JoinedOutputMetric`

   Computes the quantile of the absolute error.

   .. note::

       This class is only available on a paid version of Tumult Analytics. If you
       would like to hear more, please contact us at info@tmlt.io.

   .. note::
       This error metric is not analytic, and is calculated from a single run of the
       DP and baseline.

   How it works:
   1. Take the intersection of the groups in the DP and baseline.
   2. Compute the absoulte error for each group. The absolute error is
   abs(dp - baseline).
   3. Return the n-th quantile of the absolute error.


   .. rubric:: Example

   ..
       >>> import pandas as pd
       >>> from pyspark.sql import SparkSession
       >>> spark = SparkSession.builder.getOrCreate()
       >>> dp_df = spark.createDataFrame(
       ...     pd.DataFrame(
       ...         {
       ...             "A": ["a1", "a1", "a2"],
       ...             "X": [50, 110, 100]
       ...         }
       ...     )
       ... )
       >>> dp_outputs = {"books_by_zip_code": dp_df}
       >>> baseline_df = spark.createDataFrame(
       ...     pd.DataFrame(
       ...         {
       ...             "A": ["a1", "a1", "a2"],
       ...             "X": [100, 100, 100]
       ...         }
       ...     )
       ... )
       >>> baseline_outputs = {"books_by_zip_code": baseline_df}

   >>> # Create the metric
   >>> quantile_ae_X = QuantileAbsoluteError(
   ...     output="books_by_zip_code",
   ...     measure_column="X",
   ...     quantile=0.5,
   ...     join_columns=["A"]
   ... )
   >>> quantile_ae_X.quantile
   0.5
   >>> quantile_ae_X.join_columns
   ['A']
   >>> quantile_ae_X.compute_for_baseline(dp_outputs, baseline_outputs)
   10.0
