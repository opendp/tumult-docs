.. py:class:: Tunable

   Named placeholder for a single input to a :class:`~.SessionProgramTunerBuilder`.

   When a :class:`~.Tunable` is passed to a :class:`~.SessionProgramTunerBuilder`, it
   is replaced with the concrete values for the tunable parameter when building
   :class:`~.SessionProgram` s inside of methods like
   :meth:`~.tuner.SessionProgramTuner.error_report` and
   :meth:`~.tuner.SessionProgramTuner.multi_error_report`.





   .. py:attribute:: name
      :annotation: :str

      Name of the tunable parameter.

.. py:class:: SessionProgramTuner(builder)

   Base class for defining a object to tune inputs to a :class:`~.SessionProgram`.

   .. note::

       This class is only available on a paid version of Tumult Analytics. If you
       would like to hear more, please contact us at info@tmlt.io.

   SessionProgramTuners should not be directly constructed. Instead, they should be
   created using the :class:`~.SessionProgramTunerBuilder` that is automatically
   generated for each subclass of :class:`~.SessionProgramTuner` which can be accessed
   using the `Builder` attribute of the subclass.





   .. py:attribute:: baseline_options
      :annotation: :Optional[Union[Dict[str, tmlt.analytics.non_private_session.NonPrivateSession.Options], tmlt.analytics.non_private_session.NonPrivateSession.Options]]

      Configuration for how baseline outputs are computed.

      By default, a SessionProgramTuner computes both the DP outputs and the baseline
      outputs for a SessionProgram to compute metrics. The baseline outputs are computed
      by calling the :meth:`~.session_interaction` method with a
      :class:`~.NonPrivateSession`. The `baseline_options` attribute allows you to
      specify different configurations for the :class:`~.NonPrivateSession` used to
      compute the baseline. You can also specify multiple configurations for computing
      the baselines with different options. When multiple baseline configurations are
      specified, the metrics are computed with respect to each of the baseline
      configuration (unless specified otherwise in the metric definitions).

      To override the default baseline options
      (see :class:`~.NonPrivateSession.Options`), you can set this to
      :class:`~.NonPrivateSession.Options`.

      If you want to specify multiple baseline configurations, you can set this to a
      dictionary mapping baseline names to :class:`~.NonPrivateSession.Options`.

   .. py:attribute:: metrics
      :annotation: :Optional[List[tmlt.analytics.metrics.Metric]]

      A list of metrics to compute with the `error_report` method.

   .. py:attribute:: Builder
      :annotation: :Type[SessionProgramTunerBuilder]

      The builder for a specific subclass of SessionProgramTuner.

   .. py:attribute:: program
      :annotation: :Type[tmlt.analytics.program.SessionProgram]

      SessionProgram to be tuned.
