class NoPrivacyQueryExprCompiler(enforce_keysets, enforce_clamping_bounds, enforce_constraints, enforce_flat_map_truncation, enforce_private_join_truncation, enforce_suppression, output_measure=PureDP())#

Bases: tmlt.analytics._query_expr_compiler.QueryExprCompiler

Compiles a list of query expressions to a single measurement object.

Requires that each query is a groupby-aggregation on a sequence of transformations on a PrivateSource or PrivateView. If there is a PrivateView, the stability of the view is handled when the noise scale is calculated.

A NoPrivacyQueryExprCompiler object compiles a list of QueryExpr objects into a single object (based on the privacy framework). The Measurement object can be run with a private data source to obtain non-noisy answers to supplied queries.

Parameters

  • enforce_keysets (boolbool) – If False, evaluate will ignore keysets provided by the user, as well as keysets generated automatically using get_groups. Instead, the values which actually appear in the data will be used as the keyset.

  • enforce_clamping_bounds (boolbool) – If False, evaluate will ignore clamping bounds provided by the user, and will instead not apply any clamping bounds.

  • enforce_constraints (boolbool) – If False, evaluate will not apply any constraints provided to enforce().

  • enforce_flat_map_truncation (boolbool) – If False, evaluate will ignore the max_rows value provided to flat_map() and not apply any truncation.

  • enforce_private_join_truncation (boolbool) – If False, evaluate will ignore the truncation strategies and parameters provided to join_private(), and will not apply any truncation.

  • enforce_suppression (boolbool) – If False, when evaluating a SuppressAggregates query, no suppression will be applied - the results of the original aggregation query will be returned.

  • output_measure (PureDP | ApproxDP | RhoZCDPUnion[PureDP, ApproxDP, RhoZCDP] (default: PureDP())) – Distance measure for measurement’s output.
