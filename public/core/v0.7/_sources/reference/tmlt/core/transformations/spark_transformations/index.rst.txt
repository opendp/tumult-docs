


****************************
spark_transformations
****************************

.. py:module:: tmlt.core.transformations.spark_transformations

.. autoapi-nested-parse::

   Transformations for manipulating Spark DataFrames.


Modules
=======

.. list-table::

   * - :mod:`.add_remove_keys`
     - Transformations that transform dictionaries using :class:`~.AddRemoveKeys`.
   * - :mod:`.agg`
     - Transformations for grouping and aggregating Spark DataFrames.
   * - :mod:`.filter`
     - Transformations for filtering Spark DataFrames.
   * - :mod:`.groupby`
     - Transformations for performing groupby on Spark DataFrames.
   * - :mod:`.id`
     - Add a column containing a unique id for each row in a Spark DataFrame.
   * - :mod:`.join`
     - Transformations for joining Spark DataFrames.
   * - :mod:`.map`
     - Transformations for applying user defined maps to Spark DataFrames.
   * - :mod:`.nan`
     - Transformations to drop or replace NaNs, nulls, and infs in Spark DataFrames.
   * - :mod:`.partition`
     - Transformations for partitioning Spark DataFrames.
   * - :mod:`.persist`
     - Transformations for persisting and un-persisting Spark DataFrames.
   * - :mod:`.rename`
     - Transformations for renaming Spark DataFrame columns.
   * - :mod:`.select`
     - Transformations for selecting columns from Spark DataFrames.
   * - :mod:`.truncation`
     - Transformations for truncating Spark DataFrames.
