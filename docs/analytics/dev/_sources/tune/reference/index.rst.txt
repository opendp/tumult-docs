Tumult Tune API reference
=========================

.. note:: 

    |PRO_NOTE|

The Tumult Tune API reference is organized as follows.

* :ref:`Tuning programs<tuners>` introduces the :class:`~tmlt.tune.SessionProgramTuner`,
  the main class used to generate error reports. It also lists the various ways that
  users can interact with error reports.
* :ref:`Defining error metrics<metrics>` lists the built-in error metrics provided by
  Tumult Tune, and the methods and classes used to define custom error metrics.
