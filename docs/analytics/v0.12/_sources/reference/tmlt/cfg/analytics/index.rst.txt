.. py:data:: license_file_path_override

   License path variable.

   Update "PATH_TO_FILE.json" to be the absolute path of ypur Tumult Analytics Pro license file.

   .. code-block:: python

       from tmlt.cfg import analytics
       analytics.license_file_override = "PATH_TO_FILE.json"
       import tmlt.analytics
