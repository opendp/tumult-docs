Tumult Analytics tutorials
==========================

These tutorials introduce the key concepts of the Tumult Analytics API, and
help you design and implement your first differentially private data
publication.

1. **Interface basics and first queries** introduces the fundamentals of Tumult
   Analytics, and explains how to compute simple differentially private
   aggregations.

   .. toctree::
      :maxdepth: 1

      first-steps
      privacy-budget-basics
      clamping-bounds
      groupby-queries

   |

2. **Advanced queries and privacy notions** demonstrates how to write queries that
   include data transformations, and how to use Tumult Analytics in contexts where a
   single person can contribute to multiple rows in the input data.

   .. toctree::
      :maxdepth: 1

      simple-transformations
      privacy-id-basics
      more-with-privacy-ids
