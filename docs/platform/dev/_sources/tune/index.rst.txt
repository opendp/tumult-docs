Tumult Tune documentation
=========================

.. note:: 

    |PRO_NOTE|

Tumult Tune allows users to understand and optimize the privacy-utility
trade-offs of differentially private programs.

New users should start by learning how to
:ref:`write simple programs<first-steps>` with Tumult Analytics, then follow the
:ref:`Tumult Tune tutorials<error-measurement-basics>`.
