.. py:function:: gaussian_inverse_cdf(u, sigma_squared, p, prec)

   Returns inverse CDF for N(u,sigma_squared) at p.

   :param u: The mean of the distribution. Must be finite and non-nan.
   :param sigma_squared: The variance of the distribution.
                         Must be finite, non-nan and non-negative.
   :param p: Probability to compute the CDF at.
   :param prec: Precision to use for computing CDF.



.. py:function:: gaussian(sigma_squared, u = 0, step_size = 63)

   Samples a float from the Gaussian distribution.

   In particular, this returns a sample from the Gaussian
       :math:`\mathcal{N}_{\mathbb{Z}}(u, sigma\_squared)`

   :param sigma_squared: The variance of the distribution.
                         Must be positive, finite and non-nan.
   :param u: The mean of the distribution. Must be finite and non-nan. Defaults to 0
   :param step_size: How many bits of probability to sample at a time.
