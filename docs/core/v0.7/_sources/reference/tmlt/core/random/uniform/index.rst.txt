.. py:function:: uniform_inverse_cdf(l, u, p, prec)

   :param l: Lower bound for the uniform distribution.

   :param p: Probability to compute the inverse CDF at.
   :param prec: Precision to compute the CDF with.



.. py:function:: uniform(lower, upper, step_size = 63)

   :param step_size: Number of bits to sampler per iteration.
